### 社会行动理论
* 对行动做诠释性理解，对整个社会结构和过程做出因果性说明
* 个体，个人及其行动 = 社会行动(做给别人看的，产生影响) = 逻辑起点
**任务**：主观意义解释行动 -> 特定类型的行动变的可理解 `因为神秘/封建/家产官僚制下，人的行动`
* 分析框架
    * 行动者，目标，特定情境(工具，限制，条件)，规范与价值
    * 研究人的行为 -> 社会的行为

* 两个方向：文化解释和环境 -> 分析动机/主观意义(规范(受到惩罚/自律) + 价值) + 制度分析
* 社会行动分类：
    * 情感，传统
    * 价值合理(审美，意识形态(苗x 草√)，政治，)，工具合理(黑猫白猫)
    * (非理性 -> 理性) -> 社会结构+分层

### 宗教社会学

* 宗教：(终极条件/超自然力量)做出价值判断
* 宗教信仰与人们社会行动的关系，影响到的经济行为(为什么相信，以及如何影响行为)
* 价值中立

**新教伦理与资本主义精神**
* (新教)天职观:重视世俗职业 vs. 排斥现事:修道院
    * 伦理理性化 -> 行为理性化 (讲条理，形成系统的生活方式)

* 预定论：证明自己是上帝的选民
* 禁欲：扩大再生产

* 反证，犹太教没有首先出现资本主义
    * 伦理的二元性(本邦和外邦) vs 普遍主义(资本主义)
    * 贱民性格，自我隔离状态


**中国的宗教**
* 制度分析(法律，经济，行政) + 文化分析(儒教，道教)
    * 伦理本位(西方，认识本位)，做人比做事重要
    * 儒家，适应世界，传统主义(西方，掌控世界)。道家，迷信巫术
    * 迷信巫术，传统主义 = 神秘主义(神人合一，跳大神)

* 制度障碍：
    * 结构性阻力(中央集权 + 家产官僚制)，高度统一，排斥竞争 
    * 自古的货币观念很重，经济观念受传统主义约束
    * 货币很早，币值不统一，漕运和交易的银两不同
    * 城市形态，侧重行政/军事 -> 政令不及于县，没有经济功能(希腊城邦自治)
    * 行会制度，
    * 农村血缘，家族构成真正的自治单位，难以统一治理
    * 伦理本位，入世。理性主义+神秘主义
    * 人本主义(人类中心,)，拟人化人类中心，通过一种人格化认识世界。阻碍了知识本位，没有客观的研究。以有道德的人为中心。

> 学到神秘主义，先前看不懂的就有答案了。为什么有人睁眼说瞎话还能成功？为什么要攻击受害者？比如买到烂尾楼反而被讲活该，甚至直播被打。答案是，只有工具理性，而没有伦理(制度)理性化。迷信权威，知道谁好欺负，谁能被欺负。而不是公平，公正原则下的理性化。所以你看起来想屁股上的人，那就是牛逼，就有孝子替你辩经。

> 马督公评论明末时，聊到名族主义，认为中国没有名族主义，只有巴甫洛夫的狗。看见关键字就喷口水，见到权力就流口水。同民族遇到问题时，一点不愿意出头，比如河南的亮亮丽君夫妇。

### 社会科学方法论
* 实证哲学：规则性知识(必然性) -> 意志论(权力意志，非理性)
    * 背景：社科与自然科学不分家
    * 社会类比为生物(人)体
    * 独一无二性(复杂度过高)
