## 压力（紧张）
### 压力的定义
* 压力是不可避免的普遍现象`焦虑就像一个时刻陪伴你的朋友，告诉你危险，告诉你要行动`
* 人类能够反复回忆过去或预想未来，从而产生长期压力`睡觉时，还在想工作`
* 压力反应的机制
  * 引发战斗，逃跑或僵硬的反应
  * 副交感神经系统帮助恢复平静

> 看到老板，小美，小帅的时候也会突然紧张。假设现在是拍电影场景，让你模仿你自己，一样会无所适从，路都不会走了，这当然是压力。处理不好，开始抽搐了，就严重了。

### 有害的压力
* 一种描述慢性压力的术语**稳态负荷** `压力有个基础值，人无远虑，必有近忧。一般情况下是管理的好的`
* 慢性压力的特点
  * 身体`损害心血管系统`
  * 思维`加剧情绪敏感度,形成悲观视角`
  * 行为
    * 对他人表现出敌意或愤怒
    * 非理性焦虑：逃避本无需回避的事物 `错过了很多机会`
    * 酗酒`过度摄入糖分也算，冰箱里的可乐一瓶又一瓶`
    * 关系恶化：指责、贬低或消极沟通`素质很低`

* 心理健康的新认知：双维度模型
  * 压力维度：压力或痛苦的程度
  * 幸福维度：生活质量和幸福

> 即便顶着巨大的压力，也可以生活的幸福。当爱你的人，情绪崩溃的时候，说出一些过分的话，并不会在意，这是你的智慧，也是对方的幸福。

## 价值观
*当人生拥有目标与意义时，我们便能战胜逆境，在挫折后重整旗鼓*

### 价值观的定义
* 价值观是关于你想要成为一个什么样的人, 做生命中最重要的事情

* 一个换尿不湿的例子
  * 这件事本身是一件烦人的事情
  * 你可以把这件事当作成为贴心和有爱的家长的必经之路

* 价值观 vs 目标
  * 价值观 = 指南针（持续方向）
  * 目标 = 沿途里程碑（可完成项）

* 价值观的作用
  * 指导日常行为：解释为何锻炼、如何饮食、职场问候`你是奋斗逼，还是躺平狗`
  * 辅助决策：在冲突或两难中，通过自问"此刻什么对我最重要？"
  * 转化行为意义：换尿布转换为爱的实践

> 两个案例
```
1. 新工作不打卡，当空闲的时候，为什么起床去公司就成了问题。(通勤10分钟，可忽略。居家办公到底在干嘛懂得都懂)。为了完成任务 vs 因为我有想做的事情（终身学习）
2. 别人的原因，导致可能面对出差，我自然很抗拒。文哥的提问是："这个Bug是不是你own, 你想不想解决，谁的原因能解决问题吗？" 让我冷静下来了。
```

* 人类极易偏离价值观
  * 决策能力下降
  * 情绪化反应增多
  * 行为与价值观背离 → 事后懊悔 → 压力循环加剧

* 悬崖边拔河
  * 拼命拉扯 → 但对抗不可战胜的压力只会耗尽精力
  * 接纳压力必然存在，转而专注价值导向的行为

### 价值观澄清
* 价值陈述
* **价值领域（value domains）**
  * 决定人生中扮演的角色`父亲，子女，兄弟`
  * 每个角色代表一个价值领域`我想成为怎样的父亲？`
  * 自我关怀（self-care）

* **行为落实**
  1. 确定领域`开心果`
  2. 制定价值陈述：明确该领域核心追求`团队氛围，士气高`
  3. 规划具体行为`鼓励，打破严肃氛围，情绪调动，小活动`

* **承诺践行**
  1. 定期承诺检视：识别阻碍因素，反思行为是否一致性
  2. 坚守承诺：即使遭遇压力或消极情绪，仍持续价值导向的行为

### 靶心活动（Bull's Eye Activity）
1. **澄清价值领域**
   1. 工作：该领域对你最重要的核心价值
   2. 人际：家人、朋友、同事
   3. 个人成长与健康：关于自我发展
   4. 休闲娱乐：放松与乐趣

2. 自我评估定位：高度一致/较大差距
3. 识别障碍因素
4. 制定行动计划

>举个例子
Q: 面对女孩放风筝怎么办？
A: 你为什么喜欢她？你就死活离不开她？
Q: 对
A: 这个世界上，不一定所有人都幸福，你愿意成为那个不幸福的人吗？你为了追求自己的幸福就要抛弃那些让你幸福的事情

> 每件商品，你要付费两次。付出行动是最难的部分，做的时候总有一堆理由，尤其是在做新鲜事时，不知道怎么做。真做的时候又做过度，受伤了又退缩。要真正的空出时间，留够精力，大胆开始，小心实践，快速迭代。

## 意识

### 无意识
1. 自动导航模式：完全不记得过程
2. 思维即事实谬误：brain farts，将随机涌现的念头当作绝对真理`这点在刚开始阅读时尤为明显，读书笔记和书完全不搭嘎`
3. 情绪化反应
4. 过分焦虑：沉溺过去/担忧未来，忽视眼前发生的事
5. 多任务陷阱`边玩，边思考工作/边看电视，边做事情`

### 正念
* 正念：通过有意识地、不加评判地关注当下而升起的觉察
* 正念通过提升觉察力采取最有效的行动，选择健康且高效的反应方式
* 正念技能的结构
  1. 目的性觉察（Purposeful Awareness）
     1. 对内觉察：专注感知内在体验（思绪、情绪、身体感受）
     2. 对外觉察：观察环境与他人互动
  2. 开放温和的态度（Open and Gentle Attitude）
     1. 善待自己的体验，"注意到它，然后放下"
     2. 需刻意培养非评判态度（天生习惯评判）
  3. 意向性（Intentionality）
     1. 自主选择最有效行动

* 正念技能的核心功能`根据需要应用`
  1.  获得思维清晰度
  2.  日常减压

* 正念练习
  1. 正念呼吸（Mindful Breathing）
     1. 锚定当下的天然工具：呼吸
     2. 通过"轮辐"（spokes）将注意力导向：感官体验（所见/所闻）和他人共情（所思/所感）
     3. 对走神不评判
  2. 即时暂停（Mindful Stop）`大头控制小头，脑内屁多想想`
  3. 日常活动正念化`给无意识行为增加认知`
  4. 正念饮食
  5. 通勤正念

> 正念练习固然好，但是也变的越来越敏感了。正念饮食，让我发现高中的大锅饭是真难吃，绿豆汤一股子味儿，刷牙让我学会巴氏刷牙，相应的洗漱时间变长。高敏感度，容易没有安全感，行动延迟等别的问题。

## 注意力
### 关注积极面与感恩练习

* 注意力:占据心智的主导力量,选择性地照亮特定意识内容
  * 优先劫持注意力处理威胁
  * 消极偏好:易看到自己不足,关注他人过失
  * 通过前额叶皮层的刻意调控，可突破消极偏好

* 消极注意力的倾向实际上影响着我们的感受，我们从生活中获得的意义，最终影响着我们在生活中的行为

* 我们无法控制什么会吸引注意力，但通过觉察，能够意识到我们在关注消极的事情，并且驱使我们注意一些积极的事情和令人开心的事情

* 重塑注意力的方法
  1. 寻找生机（Silver Lining）：交通堵塞时注意窗外美景
  2. 发现他人优点：寻找欣赏/钦佩的特质
  3. 建立积极锚点：可随时调用的积极记忆

> 工作忙的时候骂街，骂老板压榨你。工作闲的时候，叫老板废物拉不来业务。工作本来就应该是有忙有空闲，享受积极解决问题的努力，享受空闲时光，就是注意力机制的调整。

### 感恩
* 感恩：通过特定方式关注生活中的积极事件而产生的情感体验
  * 惊叹感（awe）
  * 发自内心的感激（thankfulness）
  * 深层欣赏（deep appreciation）

* 感恩的深度加工
  * 简单列举感恩事项
  * 问几个更深入的问题：为什么值得感恩？
    * 缺席想象法：如果没有会如何
    * 具身化写作：吃饱了，玩痛快了

* 感恩实践
  1. 每周感恩日记
     1. 列出5件感恩事项
     2. 对每项进行深层加工
  2. 感恩便签（Thank-You Notes）`认知重评（cognitive reappraisal）转化负面体验`
     1. 致物品
     2. 致挑战
     3. 致自己
  3. 感恩信与拜访（Most Evidence-Based）

> 感谢信，我看成功学，一个是卖车的，每年写贺卡。一个是零售商好像，精心挑选伴手礼。都实践过，出去玩带了特产，很快还收到了回礼，但是后面不知道怎么处理。贺卡是一起玩，还收到了回贺卡，但是并不真的懂感谢，理解人际的常态（没有在乎你）已经是很后面的事了，也没处理好，就搁置了。


## 情绪

### 情绪管理
* 情绪的构成要素
  1. 主观体验：生理反应（心跳加速） + 认知反应（产生的想法）
  2. 环境反馈：对真实或想象事件的反应
  3. 行为驱动：直接推动行动

* 情绪的功能
  1. 能量
     * 否定式：窄化注意力与行为
       * 焦虑：将注意力限制到解决当前的问题
       * 愤怒：保护和防御
     * 肯定式：拓宽注意力与行为可能性
       * 自负：更多的自信，更可能去承担更多的事情
  2. 环境警报`焦虑`
  3. 记忆强化`重大灾难的细节`
  4. 生命调味剂

* 情绪管理的目标是防止情绪升级到损害生活功能
* 判断情绪是否与情境相符
*洗手行为*
```
正常：握手后觉得恶心，简单洗手。
需干预：因害怕细菌反复洗手30分钟以上（强迫行为）
```

### 管理消极情绪
* 杏仁核劫持：当强烈消极情绪出现时，它会接管理性思维，导致我们做出狭隘、冲动的反应
* 情绪失调：每个人都有"情绪按钮"，情绪失调者会陷入危机状态
* 关键隐喻
  1. 山体滑坡隐喻: 在滑坡初期及时干预
  2. 骑象人隐喻: 仅靠意志力（骑象人）难以对抗本能（大象）,培养习惯，合作实现目标

* 管理情绪的关键步骤：识别你的"情绪按钮"
  1. 大按钮：引发强烈情绪：明确自己的"大按钮"，才能有针对性地运用
     1. 特定人的言行
     2. 常规被打破
     3. 与金钱相关的压力
     4. 对过往经历的回忆
  2. 小按钮：轻微困扰

* 调节强烈消极情绪
  1. 命名消极情绪以驯服它
     1. 识别与命名：用语言描述当前情绪,我现在感到愤怒
     2. 抽离与接纳: 它终将过去
  2. 触发放松反应
     1. 感官引导想象:想象一个让你感到快乐或平静的场景
     2. 渐进式肌肉放松:感受张力释放带来的平静
     3. 深呼吸法: 关闭“战斗或逃跑”反应
  3. 分心技巧
     1. 从事其他活动，如阅读、运动
     2. 为他人做一件事，如帮助同事
     3. 想象更艰难的处境
     4. 做与当前情绪相反的事
     5. 在想象中“推开”情绪`钢铁侠的手掌炮`
     6. 插入积极回忆`我想起了高兴的事`
     7. 用强烈感官刺激转移注意力`拿个冰块放到你的手里`
  4. 自我安抚技巧`心要你哄`

### 培养积极情绪
* 积极情绪的效果
  1. 拓展效应：拓宽人们的注意力和思维方式
  2. 建构效应：提升幸福感
  3. 韧性效应
  4. 抵消效应：抵消消极情绪的影响

* 培养积极情绪的方法
  1. 关注生活中的小事：生活中的微小积极瞬间
  2. 为他人付出
  3. 投入“心流”活动：沉浸于某项活动
  4. 与他人建立有意义的联结
  5. 练习感恩

* 积极情绪与消极情绪的“韧性比例”应约为3:1

> 我很早就知道情绪，是从摇滚乐开始的，摇滚乐就是愤怒，从自卑的苦情歌里带出来。后来还遇上了Hip-pop，中国新说唱，一边唱歌，一边比手势。你学MJ跳舞就知道了，会产生有很强的信心感觉和良好的感觉，然后还能带动其他人的情绪。慢慢的社交能力也有了增长，有时候为了激励自己做事，会有意识的加快呼吸，从生理上唤醒愤怒，来促使自己行动。但是全面的情绪，场景训练并没有做过。而且因为恐惧失去了很多情绪体验。其实战斗-逃跑就是情绪体验的一种，怎么面对老板，异性，客户，都是要一点点积累的经验，最近在努力的实践。

## 治愈式生活方式
* 我们对生活质量的满足取决于我们每天所做的选择，这些选择共同作用可优化身心状态
* 生活方式定义：生活价值观、信念、习惯与实践构成的整体
* 现代社会的挑战：
  * 物质便利`手机严重影响睡眠`
  * "内卷化竞争"`久坐`
  * 多任务处理导致压力累积

* 任何时候发现生活方式不健康，都可以通过**微小渐进**步骤引入TLCs

### 大三项
#### **运动锻炼**
* 缓解抑郁、焦虑，提升自尊与人生展望`这是真的`
* 美国CDC指南
  * 有氧运动: 每周≥2小时中等强度（如快走、骑自行车、水中有氧）
  * 力量训练：每周≥2天（如举重、阻力带训练、瑜伽）
* 克服不锻炼的借口
  * 聚焦行为长期价值
  * 将阻碍想法标记为"只是 passing thoughts（路过念头）"

> 不是到健身房的运动才叫运动，平时走路，有意识的太高大腿，10分钟左右的时间，就可以进入有氧(实践过的)。平时就注意体态问题，不要高低腿站立

> 体育是一个天生的需要批判性思维的学科。因为本身就是依据统计学，很多合适的东西并不适合你。必须要十分的相信自己的感觉。基本的思路是：

```
1. 基础的调研，学习单项运动的基础知识
2. 谨慎少量的实验，发现问题，及时关注

Q: 我害怕受伤，要学的东西太多了，似乎怎么准备都不充分，个体的差异巨大。
A: 个体差异巨大，偏移标准并不意味错误，出现问题(疼痛/不适)才叫错误，而不是纸面参数
A2: 很多知识有依赖条件，你必须尝试之后才能学会合适的内容
A3：受伤是一定的，控制到能恢复的程度，下次会变得更强
A4: 不收到反馈，等于没有进步，一定会进入虚无状态
```

#### **睡眠**
* 双重调节系统`两者协调`
  1. 睡眠驱力（Sleep Drive）：类似饥饿感的生理需求
  2. 生物钟（Circadian Rhythm）：内在计时机制

* 睡眠剥夺的后果
  1. 认知功能: 记忆,反应速度
  2. 情绪调节：易怒
  3. 生理健康：免疫功能抑制

* 优质睡眠管理
  1.  睡眠卫生（Sleep Hygiene）
      1.   固定作息`严格控制起床时间，睡得晚也不要补，从而控制睡觉时间`
      2.   仅当困倦时上床（若20分钟未入睡，应起身进行低刺激活动）
      3.   避免午睡
  2.  刺激控制（Stimulus Control）
      1.  仅用于睡眠与亲密关系（消除工作/娱乐关联）`移除电子设备`
      2.  物理参数：温度，光线
  3.  认知亢进管理（Cognitive Hyperactivity）
      1.  躺下后思维反刍（rumination）加剧
          1.  正念呼吸冥想（Mindful Breathing）
          2.  "忧虑时间"技术：白天设定专门时段处理焦虑事项
      2.  越努力睡越清醒：放宽心
  4.  **实践补充**
      1.  不想睡，明知晚了，却非常沮丧，不想睡 -> 睡前的情绪调节，正念调整
      2.  有想做的事情，推迟睡觉 -> 调整到早起来做
      3.  把睡觉当作一件美好的事，记录回忆慵懒的，精神焕发的样子，感受睡觉的美好

> 我把后果列出来，因为是真的。长期熬夜会得带状疱疹(水痘病毒)，我已经见过好几起了，医生都惊讶。自从山河四省高中推广衡水模式，熬夜就变成习惯了，20多的时候还没觉得怎么样（大学休息时间多）。25之后就是靠肾上腺素顶着。再大一点，还不改，就会长期焦虑，压抑。反正，我的体验是这样，

#### **健康饮食**
* 核心原则
  * 人如其食`山西人要午休（因为碳水）`
  * 适度平衡：减少不健康食品频率（非完全戒除）

* 饮食策略
  1. 进食方式：慢食
  2. 彩虹饮食：多蔬果
  3. 优质碳水：全谷物替代精制碳水
  4. 脂肪选择：坚果/奶酪 替代 红肉/油炸
  5. 蛋白：豆类，鱼类，鸡蛋
  6. 控糖盐`尤其是工业糖（饼干，果汁，饮料）`

> 最近在慢慢感受吃饭的乐趣，会为了吃东西跑很远。但还是觉得吃饭(慢)是件很讨厌的事情。

> 三大项其实是最重要的项，没有体力，精力，能量，再好的思考工具也没有用。而且这三项是思考，选择的具体体现，工作狂就容易牺牲睡眠时间，就更易怒。就算其他的学不会，这三大项一定得慢慢的践行起来。

## 人际关系
* 以一种积极的方式联系其他人
* 韧性关系的支柱
  1. 社会支持: 建立"支持者名单"（每周主动联系3人）
  2. 导师关系: 寻找领域专家进行月度成长对话
  3. 利他行为: 每日微小善举
  4. 榜样选择: 制作"榜样特质分析表"

* 关系质量的评估
  1. 能量平衡理论
     1. 消耗性关系（Energy-Draining） vs 滋养性关系（Energy-Giving）
  2. 三维度评估法
     1. 情感支持：是否提供安全感？
     2. 成长激励：是否推动自我突破？
     3. 互惠程度：给予与接收是否平衡？

### 社会支持
* 有效社会支持
  1. 指导型: 提供应对困境的具体建议
  2. 情感型: 共情陪伴
  3. 归属型: 创造被接纳感
  4. 愉悦型: 共同参与轻松活动
  5. 工具型: 实际事务协助

* 无效社会支持
  1. 压力竞赛`你这算啥，我更惨`
  2. 错误建议`更加糟糕了`
  3. 情感否定`你想太多了，你反应过度了`
  4. 分心回应`敷衍应答`

### 导师关系
* 导师的目标是提供无条件的积极关注，给被指导者建议和建议，帮助他们成长和发展，达到更高的水平
* 维果茨基近端发展区：提升和增进我们的技能
  * 独立能力圈
  * 潜在发展圈

* 可以提供什么？
  * 自我效能感，也就是他们认为自己能够产生某种期望的能力。
  * 导师可以把你和其他人联系起来

* 构建优质导师关系
  1. 需求定位：明确要提升的领域
  2. 人选筛选：寻找该领域资深人士
  3. 协议制定：约定指导形式/频率
  4. 定期执行：固定日程
  5. 反馈接纳：建立"成长型思维"

### 榜样

  1. 核心特质
     1. 价值观坚定性`大张伟从穷开心之后到现在就一样`
     2. 服务导向
  2. 识别误区：名人≠榜样
  3. 近身榜样
     1. 可观察`一定要频繁的接触`
     2. 可模仿
