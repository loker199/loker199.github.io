## 万有引力定律-物理定律的范例

* 引力定律·作为物理定律的一个具体的例子`而不是抽象的，含糊其辞的讲`
    * F=GMm/RR + (补充 F = ma). 足够介绍引力定律。其他的都是数学结果。

* 这条引力定律包含什么结果(发现的故事和影响)
  * A：行星绕太阳运转-> 到底是怎样环绕太阳运行？
  * 第谷：记录各个行星的精确位置，精确的将各个理论比较分辨（观察事物，详尽记录，为可能的解释提供线索）-> 跨入现代科学的钥匙
  * 开普勒：试错法分析：圆形轨道与观测数据产生巨大偏移。重新尝试得到：椭圆，单位时间面积相等、轨道大小与环绕时间
  * A: -> 什么使得行星环绕太阳运动
  * 伽利略研究运动规律并进行大量实验(球体沿着斜面下滑，单摆往复) ：惯性定律+理想化条件
  * A: -> 当物体不沿直线前进会发生什么？
  * 牛顿：力改变物体的速度，速度变化有多快。带入行星，需要一个下坠的力，就是太阳。（没人知道为什么物体要向前，和惯性定律起源未知）。同时使用该假设推导出开普勒三定律。牛顿没有表达新观点，用另一种语言陈述了开普勒定律
  * A: -> 将想法推广：木星的卫星，月球与地球。和拉你在地面的是同一种力么？
  * 比较月球一秒的下坠距离和伽利略算出的地球上一秒的下坠距离 -> 两个毫不相关的事情联系在一起 -> 现象得到解释：潮汐
  * A: -> 测量精度提高：精细检验（木星的卫星）
    * 奥莱·罗默：相信引力定律，光的传播需要时间 -> 当一条定律正确时，能够被用来发现另一条定律
    * 微小的吸引，发现海王星
    * 对水星的运行计算不完全正确 -> 修改牛顿定律
  * -> 定律的有效范围时多少？
    * 太阳系外：双星轨道
    * 卡文迪许测量微小的物体
  * -> 平方反比引起的猜想，静电力. 现在还不明白各个定律之间的关系

* 尽管说明了数学定律，但是什么是引力？为什么是这样运行的？谁要求太阳这样做的？

## 数学与物理的关系
> 我对数学的印象是变化。即，欧式几何：一个点，也是一个圆心，也是直线上的一点，有线段就能画出等边三角形，同时又是角分线上的一点。都是同一个点。代数：0 也是log1, e1-1, 极值点

* 数学在处理大规模数据、描述复杂情况的现象、赋予基本规则里有很大作用
  * 统计学 -> 平均数，新细菌群/比例
  * 完整的棋局 -> 抽象的推理转换成数学问题
  * 计算机的组合电路

* 在物理中有些需要数学来表达基本定律（牛顿的引力定律）
  * 法拉第定律 -> 不需要真正的数学知识。电子和电荷量，并不高深
  * 引力定律 -> 是数学表达，可是无法解释万有引力的机制
    * 一种解释万有引力机制的理论：（粒子撞击）
    * 对所有结果进行分析，看是否能预言其他现象
    * 目前只有牛顿定律能描述引力现象，而且不会引起其他错误
  * 深入物理不用数学是不可能的(基于以上理由)

* 数学不仅是一种描述语言，还是一种推理语言，是一些人通过仔细思考和推理产生的结果的一个大集合，通过数学可以将一个表述与另一种表述联系起来(没有数学无法解释为什么两种表述是同一回事)
  * 证明：引力指向太阳 = 行星的矢径会在相等的时间扫过相等的面积
    * 牛顿的方法
      * 未受力情况下，三角形底乘高面积相等
      * 受力情况下，微分计算，同底等高三角形面积相等
    * 代数法

* 是否存在一个出发点可以推出整个理论？是否存在更基础的表述，或是更间接的表述
  * 巴比伦传统 -> 大量例题，掌握规则，一定程度上的推理
  * 欧几里得 -> 所有的几何原理来自一组基本公理
  * 现代数学建立在公理上，效率低
  * 物理更像巴比伦传统，记住公式，效率更高
    * 力指向太阳
    * 相同时间相同面积 -> 角动量守恒 -> 在粒子层面也成立，不依赖引力定律
  * 为了理解物理学，人们总要在逻辑上保持一种灵巧的平衡，并在他们的脑中记住所有不同的命题及其相互关系。新的定律往往超出推导范围，只有在所有定律都被发现时，这种做法才不重要。

* 物理定律实际上是被精妙地构造起来的，使得定律的表述在性质上有如此大的不同，三种不同的方式表述万有引力定律，是完全等价的表述但是听起来截然不同
  1. 作用力（牛顿定律），
  2. 引力场，势
  3. 最小路径
  * 这三种数学结果都是一样的是等效的，但是当你去猜测新的定律时，种种哲学直觉都会失败，必须找出所有可能性，尝试所有选择。
    * 如果信号的速度无法超越光速，力就不是瞬时作用，牛顿定律的表达方法就变得非常不合适，场很适用
    * 量子力学中最小值原理存在，而其他两种表述不对
  * 如果你的理论结构只有部分准确，必须保又观察事物的所有方式

* 物理与数学的关系
  * 数学只在乎推理是否正确，不在乎代表什么。物理学家所说的都是有实际意义的，要转化成现实世界的事物，从而保证正确性。
  * 数学追求普遍适用（描述多维世界n=3），物理只关注具体的定律（只关注引力而不是任意的力）
  * 数学要求高精度，物理在明白一些原理之后可以跳过一些进行快速推导。
  * 物理模型(猜测/本能感觉/哲学原理)在发现新定律上式很有用的（麦克斯韦电动力学齿轮惰轮）狄拉克猜测方程式。数学式表达自然的一种深层方式，用哲学原理表达自然或者机械式感觉表达自然，都不是有效的方式。

* 最后，两种文化：有数学经验（能够欣赏自然）和没有数学经验。所有理智的论证无法把对世界的理解传达给没有数学经验的人(像和聋子交流音乐一样)

## 伟大的守恒定律|物理拼图的线索
* 守恒：存在一个数据，能够计算在某一时刻的值，在自然变化中，该值不变
* 守恒定律的作用：不许要深入观察，就可以对游戏规则有所了解`即使象棋下的非常快，存在不许要步骤也能确定的事，象不会改变棋盘的颜色`

* 电荷守恒：物体上的电荷总量是一个固定值，不会变化
  * 大金属球壳里，球壳表面接验电器，检测电荷。玻璃棒和猫毛摩擦
  * 假设世界由两种粒子组成：质子和电子。正电荷和负电荷
    * 假设电子永恒不变，不会分解和消失 -> 质子数减去电子数的值不变
    * 事实上电子数和质子数会变化(中子可以衰变质子和电子)
    * 存在另一种粒子：带正电，不是质子。遇到电子产生湮灭，辐射出光
  * 电荷守恒存在两种解释：
    * 正和负同时产生和消失
    * 某个电子的移动/定域的电荷守恒
      * 证明：在移动的飞船中点，观察两端的电荷产生
  * 电荷是一种场的来源
  * 基本单元成倍出现，计算方法是数数量

* 重子数守恒
  * 基本粒子带重子
  * 重子＋重子反应
  * 粒子出现时发生反应的概率，仅考虑快速发生的现象，奇异数

* 能量守恒
  * 母亲找小孩的积木，重量，体积
  * 能量的形式：动能，引力势能，热能，粒子存在的能量
    * 热能时粒子动能
    * 弹性能和化学能有同一来源(化学键)
  * 守恒定律在其他领域里的作用
    * 杠杆原理，来回摇摆的配重
    * 动能-球在地面移动-微观尺度上时微小的原子相互碰撞
  * 能量守恒的第一演示
    * 小鼠喂食和燃烧的能量测算
  * 两个正在研究的能量守恒案例
    * 类星体研究困难(极不可能是基本定律错误，通常是细节不够)
    * 中子与质子电子能量的计算 -> 反中微子

* 为何可以把定律推广到尚未确定的领域？
    * 如果不假设这些定律在未知领域也成立，那将会一无所知。如果发现的定律仅仅在已经完成观察的领域内，永远无法做出预言。我们总要对尚未仔细考察过的领域提出设想，否则科学就没有意义。-> 运动中的物体质量会变大
    * 无法了解所有实验，所要的条件
    * 不只是简单的描述做过的实验，要在实验的范围外提出定律才是成功，使科学充满不确定性

* 角动量守恒
  * 发电机-电磁场的守恒
  * 在量子力学是以基本单元整数倍出现

> 一个感受是，概念的解释非常少，反而是各种实验，推理，还有哲学推演。稍微走神就不知道在将什么主题。这就是懂，大量的实践案例才叫懂。

## 物理定律中的对称性
* 如何定义对称性：对一个事物操作，完成操作之后，它看起来和先前完全一致
* 空间平移对称性：
  * 只是空间上移动了位置
  * 移动所有东西(参考系)，把任何会影响实验的东西一起移走
  * 牛顿定律的对称性
    * 空间上的移动，移动太阳系，比例不会变
    * 时间上的迁移，两年前的移动规律与两年后没有区别
    * 时间移动的特例：宇宙起源之前

* 空间中的转动
  * 数学方式：复合坐标x,y
  * 匀速直线运动的对称性研究(无法区分谁的观点是对的)
    * 初始速度不影响运动规律
    * 麦克斯韦，关于运动中光速的测量(光速不变)：时间可能是个相对概念
    * 无法在匀速直线运动中讲出自己有多快 - 四维空间

* 一种对称性：同类原子替换，对任何现象不产生问题`不会穿过物体的来源`

* 对称性不成立的例子
  * 无法改变尺度，`伽利略研究杠杆和尺度，为较大的动物配置骨骼`
  * 飞船里固定的角度旋转，是能分辨出来的 `傅科摆证明地球在转动`

* 空间中的反射问题
  * 空间镜像不会产生差异：镜像的，精密的钟，一段时间后，是否能区分
    * 最先的猜想是无法区分
    * 有些东西会有差别
      * 偏振光穿越蔗糖溶液，固定的向右
        * 使用人造糖重复实验，细菌会吃掉一半，剩余可以偏左
        * 蛋白质是存在手性，可能是竞争后剩余的/偶然出现后衍生的
    * 自然界对左和右一样的原理不对
      * 中子衰变成质子，电子和反反中微子，测量电子的自旋，向左

* 守恒定律和对称性定律
  * 假设最小路径存在，左右路径的作用量是相同的

## 过去与未来（思考的等级和现象）
* 过去与未来的显著区别：某些原子的运动规律是单向的，使得世界上的所有现象似乎都只朝着一个方向发展
  * 至少目前知道的物理定律是双向的`运转轨道逆行也没关系`
  * 事实上：地球自转的速度在变慢(由于潮汐的摩擦)
  * 墨水(蓝色和清水)混合
    * 微观上分子碰撞可逆
    * 生命的一般偶然性
    * 如果一个盒子，只有几个分子，很容易分层，但存在几亿。

* 事物是什么时候开始有序的，为什么只看到部分有序的普通状况
  * 有序(确定的状态)，如果是均匀状态，则有序是异常情况
  * 世界变有序的方法(一种理论)：世界不停的演变，我们正在观察到恢复无序的状态
  * 假设：过去的宇宙更加有序，存在描述世界过去状态的定律(天文学，考古学)

* 不可逆性的性质：
  * 可利用能量持续减少
    * 观察不可逆机器如何运作，棘轮与旋叶的热交换
    * 能量总数不变，但会变得更加均匀，最终会没有单向变化(能量的单向传递)
    * 如果物体处于不同的温度(表示从一个物体获取能量的难度)，并且处于封闭环境，时间流逝，会逐渐趋于同一温度，可利用的能量持续减少(熵增定律)
  * 时间不可逆
  * 熵增定律并不直观，和经验没有直观的联系`物理定律可逆，但现象不可逆`
    * 远处观察冰川 与 冰晶是六边形组成
    * 原子核能级与 氦形成碳

* 概念的等级
  * 基本定律，近似概念的术语，并相信(期望)能解释世界`热与原子震动，盐晶`
  * 物质的性质逐渐上升层级 `偏振，表面张力，风暴，人，权术，美，恶`
  * 理解了物理学基本定律无论如何都不会让你对这个世界的意义有更好的理解，实际经验的细节，往往与基本定律想去甚远。实际上我们也做不到回溯到本质。
  * 必须关注的是事物整体结构上的相互关联，没有哪一段更接近上帝，无论站在拿一端，都只能从这一端的码头离开，并寄希望于从另一端可以获得彻底的理解，这种想法是错误的。无论是展开“恶，美，希望”一端，还是基本定律一端，期望仅凭一个方向就能够得到对整个世界的深入理解，这是错误的。两端人相互漠视也是不明智的。把一个层次与另一个关联起来，一直在提升我们对世界的理解。 


## 量子力学中的概率和不确定性
* 最初的实验观察是基于生活经验做出合理的解释，随着案例变多，对观察到的事物做出一只的描述时，一些解释就成为定律。而且这些定律会逐渐变得越来越不合常理。
  * 如果你认为两件事情同时发生，这不过是你主观的看法，同时性仅仅是一种主观的印象

* 只有通过精确地测量和细致的实验，我们才能拥有更广阔的视野，然后看到意想不到的事物，那些远远超过想象的事物，成为想象力的延申，来理解发生的事情。
  * 光的历史
    * 一簇粒子、
    * 波、
    * 两种特性（光电效应）
  * 电子
    * 粒子
    * 波（电子衍射）
  * 在量子力学方程中得到统一，行为方式并不像你之前见过的任何事物，在非常微小尺度下的行为是完全不同的，必须要有足够的想象力才能弄清楚它们是如何运动的。

* 类比粒子的行为
  * 子弹实验(发射源, 单小孔, 两个小孔, 挡板/探测器)
  * 分别打开孔N12,N1,N2,计算到达的数目
  * 更加正态分布

* 类比波的行为
  * 记录探测器振动的幅度
  * 干涉分布

* 电子/光子的情况
  * 探测器接放大器，不会同时通过N12
    * 呈现整体达到，得到的是衍射波形
  * 仅通过N1，呈现N1的粒子波形
  * 仅通过N2，呈现N2的粒子波形
  * N12≠N1+N2

* 解释
  * 一个电子，要么通过孔1，要么通过孔2 ，说法不正确(相加结果不对)
  * 在N1，N2增加探测器，观察电子经过的情况，变成了N1曲线，N2曲线，而且总数也变成N1+N2,有光照的情况，仅仅影响最终到达的情况(光照影响了曲线，变的平滑)
  * 不确定原理：只能假设我们的实验能力存在一些基本性的局限，即，做实验不能想多精细就做多精细

* 目前的观点：当你观察时，就是确定的。不观察时，就是不确定的。当你不观察时，就做预言就会出现错误 

* 概率的存在可以回溯到基本定律(不同于复杂情况下，使用概率进行简化计算)，即物理的基本定律中存在概率，物理学在这方面已经有点放弃了形式逻辑体系. 我们无法做出预测，并不是缺乏详尽的知识（隐变量理论），在一定程度这（随机性）看上去是固有的性质。

* 什么是科学的必要条件？科学存在的必要条件是实验能力，以及诚实地报告出结果，报告的结果必须不受某些人说他们喜欢结果是什么样的影响. 最后很重要的是，解释得到结果的智慧。而这种智慧在于，不应当提前确信某些事一定会发生，若仅仅带有偏见不要紧，因为如果事实为真，那么实验结果会持续积累，并不断地烦扰你，直到它们不能再被忽视。只有事先完全确信科学要有预设条件，才能忽略这些实验积累，必要条件是：大自然没有预设条件`固有的不确定性存在是事实，无法规避，无法观察`


## 寻找新定律
  我们认为我们知道什么，有什么可以猜测以及如果去猜测。
  原理加起来也无法说明大自然内容是什么，因此介绍一些东西
  物质：居然都一样，到处都是一样的原子（原子核+电子）。我们自认为知道的那些构建世界的零件：电子、光子、引力 |||| 质子、中子。足够解释低能量现象，如果解释不了那说明现代物理学是错误的（占卜、星象）
  介于中子和质子之间的作用力的没有完全理解（无法计算），高能量的质子和中子撞击，产生了非常多的粒子（可以分成很多家族），再加上目前的定律会产生无穷大的结果，或者矛盾的结果（能量为负、因果律、概率总和）

## 总结
First，猜想，计算，比较结果，错了就是错了（实验严谨的情况下）。
实验家做某些领域的实验，实验时可以产生出乎意料的结果让我们重新思考。只有尽快的证明我们的猜想错了，才能进步。（普通低能级实验，不知道怎么调毛病）
你无法证明一个含糊不清的理论是错误的（A kill mother因为爱不够）（近似对称性）
如何开始猜想？从所有已知的原理开始（但所有已知原理彼此并不融洽，所以有些东西必须要移除，问题不在于什么可能是错的，而在于准确找出什么能替代它）（简单类型的可能性有无数个（猜密码））
看历史上人是如何做出猜想的？牛顿（把那些与实验非常密切关系的概念整合在一起）麦克斯韦（整合定律，解决矛盾的地方）爱因斯坦（讨论定律可能存在的对称性）量子力学，薛定谔（猜方程）、海森堡（只分析可测量的量）但是物理学的历史不会自我重复，新发现会采用完全不同的方式。历史对我们没有帮助
尽可能把东西延申道我们已知范围之外（牛顿引力是为了了解行星）（章鱼触须）
近似对称性（事情本身就是复杂的，但是复杂的结果是简单的）（行星轨道圆形）
对于两个完全一样的理论（内在理念不同）计算结果也一样，我们要怎么判断哪个正确？（没办法区分，但是为了猜出新理论，这两种其实是完全不同的框架（简单变化前可能完全一致，但是变化之后就完全不同），一个优秀的理论物理学家，对一个现象的表达框架应该掌握7，8种（对定律的理解））（玛雅人计算时间）
