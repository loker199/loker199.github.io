## 万有引力定律-物理定律的范例

* 引力定律·作为物理定律的一个具体的例子`而不是抽象的，含糊其辞的讲`
    * F=GMm/RR + (补充 F = ma). 足够介绍引力定律。其他的都是数学结果。

* 这条引力定律包含什么结果(发现的故事和影响)
  * A：行星绕太阳运转-> 到底是怎样环绕太阳运行？
  * 第谷：记录各个行星的精确位置，精确的将各个理论比较分辨（观察事物，详尽记录，为可能的解释提供线索）-> 跨入现代科学的钥匙
  * 开普勒：试错法分析：圆形轨道与观测数据产生巨大偏移。重新尝试得到：椭圆，单位时间面积相等、轨道大小与环绕时间
  * A: -> 什么使得行星环绕太阳运动
  * 伽利略研究运动规律并进行大量实验(球体沿着斜面下滑，单摆往复) ：惯性定律+理想化条件
  * A: -> 当物体不沿直线前进会发生什么？
  * 牛顿：力改变物体的速度，速度变化有多快。带入行星，需要一个下坠的力，就是太阳。（没人知道为什么物体要向前，和惯性定律起源未知）。同时使用该假设推导出开普勒三定律。牛顿没有表达新观点，用另一种语言陈述了开普勒定律
  * A: -> 将想法推广：木星的卫星，月球与地球。和拉你在地面的是同一种力么？
  * 比较月球一秒的下坠距离和伽利略算出的地球上一秒的下坠距离 -> 两个毫不相关的事情联系在一起 -> 现象得到解释：潮汐
  * A: -> 测量精度提高：精细检验（木星的卫星）
    * 奥莱·罗默：相信引力定律，光的传播需要时间 -> 当一条定律正确时，能够被用来发现另一条定律
    * 微小的吸引，发现海王星
    * 对水星的运行计算不完全正确 -> 修改牛顿定律
  * -> 定律的有效范围时多少？
    * 太阳系外：双星轨道
    * 卡文迪许测量微小的物体
  * -> 平方反比引起的猜想，静电力. 现在还不明白各个定律之间的关系

* 尽管说明了数学定律，但是什么是引力？为什么是这样运行的？谁要求太阳这样做的？

## 数学与物理的关系
> 我对数学的印象是变化。即，欧式几何：一个点，也是一个圆心，也是直线上的一点，有线段就能画出等边三角形，同时又是角分线上的一点。都是同一个点。代数：0 也是log1, e1-1, 极值点

* 数学在处理大规模数据、描述复杂情况的现象、赋予基本规则里有很大作用
  * 统计学 -> 平均数，新细菌群/比例
  * 完整的棋局 -> 抽象的推理转换成数学问题
  * 计算机的组合电路

* 在物理中有些需要数学来表达基本定律（牛顿的引力定律）
  * 法拉第定律 -> 不需要真正的数学知识。电子和电荷量，并不高深
  * 引力定律 -> 是数学表达，可是无法解释万有引力的机制
    * 一种解释万有引力机制的理论：（粒子撞击）
    * 对所有结果进行分析，看是否能预言其他现象
    * 目前只有牛顿定律能描述引力现象，而且不会引起其他错误
  * 深入物理不用数学是不可能的(基于以上理由)

* 数学不仅是一种描述语言，还是一种推理语言，是一些人通过仔细思考和推理产生的结果的一个大集合，通过数学可以将一个表述与另一种表述联系起来(没有数学无法解释为什么两种表述是同一回事)
  * 证明：引力指向太阳 = 行星的矢径会在相等的时间扫过相等的面积
    * 牛顿的方法
      * 未受力情况下，三角形底乘高面积相等
      * 受力情况下，微分计算，同底等高三角形面积相等
    * 代数法

* 是否存在一个出发点可以推出整个理论？是否存在更基础的表述，或是更间接的表述
  * 巴比伦传统 -> 大量例题，掌握规则，一定程度上的推理
  * 欧几里得 -> 所有的几何原理来自一组基本公理
  * 现代数学建立在公理上，效率低
  * 物理更像巴比伦传统，记住公式，效率更高
    * 力指向太阳
    * 相同时间相同面积 -> 角动量守恒 -> 在粒子层面也成立，不依赖引力定律
  * 为了理解物理学，人们总要在逻辑上保持一种灵巧的平衡，并在他们的脑中记住所有不同的命题及其相互关系。新的定律往往超出推导范围，只有在所有定律都被发现时，这种做法才不重要。

* 物理定律实际上是被精妙地构造起来的，使得定律的表述在性质上有如此大的不同，三种不同的方式表述万有引力定律，是完全等价的表述但是听起来截然不同
  1. 作用力（牛顿定律），
  2. 引力场，势
  3. 最小路径
  * 这三种数学结果都是一样的是等效的，但是当你去猜测新的定律时，种种哲学直觉都会失败，必须找出所有可能性，尝试所有选择。
    * 如果信号的速度无法超越光速，力就不是瞬时作用，牛顿定律的表达方法就变得非常不合适，场很适用
    * 量子力学中最小值原理存在，而其他两种表述不对
  * 如果你的理论结构只有部分准确，必须保又观察事物的所有方式

* 物理与数学的关系
  * 数学只在乎推理是否正确，不在乎代表什么。物理学家所说的都是有实际意义的，要转化成现实世界的事物，从而保证正确性。
  * 数学追求普遍适用（描述多维世界n=3），物理只关注具体的定律（只关注引力而不是任意的力）
  * 数学要求高精度，物理在明白一些原理之后可以跳过一些进行快速推导。
  * 物理模型(猜测/本能感觉/哲学原理)在发现新定律上式很有用的（麦克斯韦电动力学齿轮惰轮）狄拉克猜测方程式。数学式表达自然的一种深层方式，用哲学原理表达自然或者机械式感觉表达自然，都不是有效的方式。

* 最后，两种文化：有数学经验（能够欣赏自然）和没有数学经验。所有理智的论证无法把对世界的理解传达给没有数学经验的人(像和聋子交流音乐一样)

## 伟大的守恒定律|物理拼图的线索
* 守恒：存在一个数据，能够计算在某一时刻的值，在自然变化中，该值不变
* 守恒定律的作用：不许要深入观察，就可以对游戏规则有所了解`即使象棋下的非常快，存在不许要步骤也能确定的事，象不会改变棋盘的颜色`

* 电荷守恒：物体上的电荷总量是一个固定值，不会变化
  * 大金属球壳里，球壳表面接验电器，检测电荷。玻璃棒和猫毛摩擦
  * 假设世界由两种粒子组成：质子和电子。正电荷和负电荷
    * 假设电子永恒不变，不会分解和消失 -> 质子数减去电子数的值不变
    * 事实上电子数和质子数会变化(中子可以衰变质子和电子)
    * 存在另一种粒子：带正电，不是质子。遇到电子产生湮灭，辐射出光
  * 电荷守恒存在两种解释：
    * 正和负同时产生和消失
    * 某个电子的移动/定域的电荷守恒
      * 证明：在移动的飞船中点，观察两端的电荷产生
  * 电荷是一种场的来源
  * 基本单元成倍出现，计算方法是数数量

* 重子数守恒
  * 基本粒子带重子
  * 重子＋重子反应
  * 粒子出现时发生反应的概率，仅考虑快速发生的现象，奇异数

* 能量守恒
  * 母亲找小孩的积木，重量，体积
  * 能量的形式：动能，引力势能，热能，粒子存在的能量
    * 热能时粒子动能
    * 弹性能和化学能有同一来源(化学键)
  * 守恒定律在其他领域里的作用
    * 杠杆原理，来回摇摆的配重
    * 动能-球在地面移动-微观尺度上时微小的原子相互碰撞
  * 能量守恒的第一演示
    * 小鼠喂食和燃烧的能量测算
  * 两个正在研究的能量守恒案例
    * 类星体研究困难(极不可能是基本定律错误，通常是细节不够)
    * 中子与质子电子能量的计算 -> 反中微子

* 为何可以把定律推广到尚未确定的领域？
    * 如果不假设这些定律在未知领域也成立，那将会一无所知。如果发现的定律仅仅在已经完成观察的领域内，永远无法做出预言。我们总要对尚未仔细考察过的领域提出设想，否则科学就没有意义。-> 运动中的物体质量会变大
    * 无法了解所有实验，所要的条件
    * 不只是简单的描述做过的实验，要在实验的范围外提出定律才是成功，使科学充满不确定性

* 角动量守恒
  * 发电机-电磁场的守恒
  * 在量子力学是以基本单元整数倍出现

## 物理定律中的对称性
  什么是对称：对一个事物操作，完成操作之后，它看起来完全一致
  空间平移对称性
  空间中的迁移
  时间上的迁移（不讨论宇宙起源仅对已知的一些）无差别
  空间中的转动
  沿直线做匀速运动（划出一部分世界，仍然不会有差别）
  麦克斯韦关于光的速度：时间可能是个相对概念
  关于沿直线匀速运动的对称性原理的结果：不能区分谁的观点是对的，时间叠加到空间变成四维空间（你用不同的视角观察会发现有一定的混合，有一些则与空间无关），要找到你在方程里加一些什么，能使方程的结果不变
  原子替换（存在同类原子）
  空间中的反射（镜子和Chinese copy）

## 量子力学中的概率和不确定性
最初的实验观察是基于生活经验做出合理的解释，随着案例变多，一些解释就成为定律。
只有通过精确地测量和细致的实验，我们才能拥有更广阔的视野，然后看到意想不到的事物，那些远远超过想象的事物。（相对论对于同时发生的说明）
光的历史，粒子（光电效应）、波（电子衍射）、粒子、两种特性（量子力学）行为方式并不像你之前见过的任何事物，在非常微小尺度下的行为是完全不同的。
电子的行为和光子完全一样（古怪的很一致）。
双孔实验： 
  如果发射子弹，条件（呈整数出现，结果取平均值）
  如果是水波，探测器（理想水波，测量水波携带的能量）（干涉）
  如果是电子，接收器（你不会在同一时刻听到两声嘀嗒—----电子是整体到达的）得到干涉的曲线。电子呈总体到达，就像粒子那样。但这些整体到达的概率，是如同波的强度那样被决定的。
检查电子要么通过小孔1，要么通过小孔2，或者同时穿过。实验观察在小孔后放置光源，如果开灯就是粒子的曲线，关灯就是波的曲线（光照影响了运动），如果逐渐调弱光源，逐渐表现成关灯时候的结果。你不可能在无干扰的情况下说出电子穿过哪个孔。
不确定性原理：不可能设计出任何装置，在确定电子穿过哪个孔时，于此同时，却不对电子产生足够的扰动进而破坏其干涉图样。
如果你不去观察，那么电子可能穿过两个孔中的任意一个，但是如果你去观察，那就必定在某一个孔中穿过（停止思考）
但是它实际的机制是什么样的？没人知道，人们只能描述现象，给你更多的例子。
物理学在这方面已经有点放弃了形式逻辑体系。我们无法做出预测，并不是缺乏详尽的知识（隐变量理论），在一定程度这（随机性）看上去是固有的性质。大自然自身也不知道电子要走哪条道。
那还要科学干嘛？什么是科学的必要条件？科学存在的必要条件只是实验能力，以及诚实地报告出结果，报告的结果必须不受某些人说他们喜欢结果是什么样的影响，最后很重要的是，解释得到结果的智慧。而这种智慧的中带你在于，不应当提前确信某些事一定会发生，若仅仅带有偏见不要紧，因为如果事实为真，那么实验结果会持续积累，并不断地烦扰你，直到它们不能再被忽视，大自然没有预设条件（固有的不确定性存在是事实，无法规避，无法观察）

## 总结
First，猜想，计算，比较结果，错了就是错了（实验严谨的情况下）。
实验家做某些领域的实验，实验时可以产生出乎意料的结果让我们重新思考。只有尽快的证明我们的猜想错了，才能进步。（普通低能级实验，不知道怎么调毛病）
你无法证明一个含糊不清的理论是错误的（A kill mother因为爱不够）（近似对称性）
如何开始猜想？从所有已知的原理开始（但所有已知原理彼此并不融洽，所以有些东西必须要移除，问题不在于什么可能是错的，而在于准确找出什么能替代它）（简单类型的可能性有无数个（猜密码））
看历史上人是如何做出猜想的？牛顿（把那些与实验非常密切关系的概念整合在一起）麦克斯韦（整合定律，解决矛盾的地方）爱因斯坦（讨论定律可能存在的对称性）量子力学，薛定谔（猜方程）、海森堡（只分析可测量的量）但是物理学的历史不会自我重复，新发现会采用完全不同的方式。历史对我们没有帮助
