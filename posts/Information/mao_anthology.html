毛泽东选集

选集里的内容，到现在，基本都变成了常识(我以为)。想想还是查漏补缺吧。

实践论

  • 生产活动是最基本的实践活动

    • 依赖物质的生产活动，结成的生产关系，各种形式的阶级斗争
    • 认识从低级向高级发展
    • 社会实践是对于外界认识的真理性的标准：在社会实践过程中，达到了思想中所预想的结果，认识才被证实
    • 不合客观外界的规律性就会失败，改正自己的思想使之适合于外界的规律性
    • 判定认识或理论之是否真理，不是依主观上觉得如何而定，而是依客观上社会实践的结果如何而定。真理的标准只能是社会的实践。社会的实践重复了好多遍

  • 认识的过程

    • 感觉和印象阶段(各个事物的片面)-> 不能造成深刻的概念
    • 社会实践的继续-> 产生概念，判断推理形成合理的结论
    • 认识过程一定是统一的两个阶段
      • 无论何人要认识什么事物，除了同那个事物接触，即生活于（实践于）那个事物的环境中，是没有法子解决的。理性认识依赖于感性认识
      • 能动的改造世界-> 从理性的认识到革命的实践

  • 如果这个人在这项工作中经过了一个时期，他有了这项工作的经验了，而他又是一个肯虚心体察情况的人，不是一个主观地、片面地、表面地看问题的人，他就能够自己做出应该怎样进行工作的结论，他的工作勇气也就可以大大地提高了。

  • 批评当时的两种情况

    • 人们的认识运动也应跟着推移和发展 -> 思想落后于实际的事:顽固派
    • 思想超过客观过程的一定发展阶段 -> 空谈主义

我提两个人：伽利略和牛顿。一个强调量化和实验。另一个牛顿万有引力定律在木星卫星上的实践

一个毛病，幻想通过一两次的实践，得出万能的道理。解决方法是及时的记录感受，记忆点，不做评价，总结。当达到一个milestone，可以用以验证时，再尝试整理理论。

  • 关于实践验证标准的三种不同的表达
    • 嘻唰唰是首大数据音乐(张伟在坏蛋调频自己讲的)
    • 哈吉米的翻唱就是比普通的翻唱(播放量)数据要好，唱哈吉米是对的
    • 你交友垃圾，差，就是因为你从来没有听过别人想要什么，永远是自嗨，你永远是在想自己要这个，要那个，就是不想给别人要的。市场要什么，你给他啊！市场就是喜欢团播，喜欢明星，网红，你为什么不给？

对正确性不予评估，很显然，说服(行动的)力量不同。所以，考虑书摘时，最好增加不同的表达风格。

实践看起来是常识，可很难在生活中践行。我偏好理论学习，忘记不确定原理无法通过想象认知。企图通过理论认识/学习/掌握技能，更好的是换成一个个小的实践。

反对党八股

  • 空话连篇，言之无物

    • 喜写长文章，但是没有什么内容 -> 下决心不要群众看,群众见了就摇头不肯看,欺负幼稚的人
    • 长而空不好，短而空就好吗？也不好 -> 应当禁绝一切空话 -> 主要的和首先的任务是裹脚布

  • 装腔作势，借以吓人 -> 是无赖 如何面对question？

    • 生怕人家驳，非常胆怯，于是就靠装样子吓人 vs 科学是真理，决不怕人家驳
    • 举例：“残酷斗争”和“无情打击” -> 没有分析对象 和 装腔作势，借以吓人

  • 无的放矢，不看对象

    • 举例：城墙标语,工字写成了“—ㄣ—”字 -> 书法好，但城墙上莫名奇妙 -> 发誓不要老百姓看，否则就很难得到解释
    • 如果真想做宣传，就要看对象，就要想一想自己的文章、演说、谈话、写字是给什么人看、给什么人听的，否则就等于下决心不要人看，不要人听。

  • 语言无味,像个瘪三

    • 为群众办事的，如果也不学群众的语言，那就办不好
    • 要向人民群众&古语&外国文学(“干部”两个字) 学习语言

  • 甲乙丙丁，开中药铺

    • 一篇文章充满了这些符号(列表符号,第一是大壹贰叁肆)，不提出问题，不分析问题，不解决问题，不表示赞成什么，反对什么 -> 形式主义 -> 做概念的游戏，使人不用脑筋想问题，不去思考事物的本质，而满足于甲乙丙丁的现象罗列
    • 什么叫问题? 问题就是事物的矛盾. -> 把问题提出来 -> 对于问题的两个基本方面(赞成/反对)加以大略的调查和研究 -> 解决问题
