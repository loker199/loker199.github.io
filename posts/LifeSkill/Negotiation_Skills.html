会谈技巧

心理咨询的理论与方法：会谈技巧

  • 心理治疗 vs 心理咨询
  • 建立咨询关系，帮助人自己解决问题
  • 会谈技巧，信任，接纳

基本理念

尊重

eg:

  • 高中生厌烦学习

    • 批评指责不上进，理由是将来没出息 -> 没关注到焦虑的真正原因，和表现出的焦虑
    • 耐心询问为什么厌烦 ->
      • 怎么脸色有点差
      • 有学习压力是正常的
      • 你说xxx的意思是，具体化，+复述

  • 上大学是唯一出路吗？表现出想尝试不同的人生路

    • 数落 -> 仿佛扼杀邪念 -> 强加自己的价值观给别人
    • 开放，积极的态度下感到被理解，被接纳 -> 引导了解信息，付出努力
      • 确实xx（复述），你知道怎么实现吗？
      • 深入了解，实践也是xxx(复述)

  • 打断陈述并给出帮助

    • 随意打断并不礼貌 -> 一厢情愿的帮助 -> 居高临下的态度说教
    • 展现相同的境遇，婉转的提出帮助意愿 -> 激发主动合作的愿望
      • 情绪表达(辛苦，)
      • 自我开放

  • 随意泄露私下的谈话

    • 隐私泄露
    • 笼统概括 -> 保护隐私

技术分析

  • 礼貌，姿态，不批评，不指责

  • 尊重：平等，礼貌，保持距离 = 偏向理性

  • 良好的咨询关系

    • 理念态度+动机
    • 无条件尊重
    • 最大程度的表达自己
    • 自我价值感，恢复自信
    • 中性价值体系
    • 充分信任 - 信任动力
      • 求助者的认知最重要
      • 假话的意义
    • 尊重隐私
      • 评价-干预

温暖

消除不安心理，激发合作愿望 vs. 过于热情

eg1:

  • 你知道xx社团吗？(进行超长的说明) -> 谢谢,可是有一点内向(怎么没去，就打退堂鼓了，还有我陪你) -> 先走了

技术分析

  • 试探性问题，得到超长说明，不管对方是否需要 -> 不耐烦，不满

eg2:

  • 你知道xx社团吗？(会在周末组织志愿者活动的社团) -> 周末什么时候有活动？(周末下午，花点自己的时间) ->很好，但有一点内向，不知道适不适合(担心是正常的，总有个过程。前几次我可以陪你) -> 太好了

技术分析

  • 问题逐个抛出，逐个回答，增加鼓励
  • 贴近了需求，打开了想法和顾虑(uncomfortable，concern)

汇总

  • 降低不良情绪(不安，疑惑，紧张，焦虑，犹豫) -> 被接纳，尊重(关心，细致，周到，耐心)

  • 激发合作愿望 -> 对咨询抱有期待

    • 关切的询问近况
    • 鼓励自我探索
    • 感谢密切配合

  • 如何做到温暖 -> 减少距离

    • 不能过于热情，显得强势
    • 倾听和友好认真的表情动作
    • 耐心与接纳(接受缓慢和反复)

参与

  • 澄清问题，启发，引导进行自我探索
  • 推进咨询进程

倾听

eg:

  • 表达对班主任的不满

    • 始终玩手机，不关注内容，不共情。你不都习惯了吗
    • 专注状态这么过分啊

  • 举例说明同学迟到和班主任的差劲行为

    • 打断，并做出主观猜测，没有体会到情绪，且阻塞表达和情绪他骂人了？
    • 啊？(请求解释)，真的吗，嗯(继续)，得罪全班的人(共情)

技术分析

  • 听懂直接表达的东西，发现言语，表情，动作透露出的信息，和省略的，没有表达出的内容和潜意识

  • 积极听并做出恰当的反应，是一种能力(允许安静的能力)

  • 收集资料，明确问题，原因，程度。建立良好的咨询关系

  • 发现积极光明的一面

  • 关注问题，情绪情感，关注内在体验

  • 参与，在恰当的时候，不打断思路，情绪的基础上，深入谈下去。

    • 点头，皱眉

  • 错误

    • 打断别人：阻塞表达和情绪
    • 急于下结论：扫兴，道德评价
    • 轻视求助者：你这算啥，你值得更好的感知问题的性质，转变观念

提问(倾听技巧)

eg1

  • 表达失眠的不安 -> (有想过为什么会这样吗)成绩下降 -> (太难了还是不认真)可能都有 -> (讲讲你的课堂情况)不能集中注意力 -> (你回想下最近有发生过什么事吗)感觉有人看着 -> (你什么事觉得别人在看着你)眼镜看着奇怪 ->(你觉得别人看你是这个原因？)是的

eg2

  • 表达失眠的不安 -> (是什么让你觉得不安呢)成绩下降 -> (突然下降是不是有什么原因)不能集中注意力 -> (注意力在哪里)感觉有人议论 -> (议论，挺累，怎么会这样？)新配了眼镜 -> (你是说大家觉得眼镜不好看，你好像不太喜欢这个眼镜)表达和共情

技术分析

  • 详细的反应，平等中立，不应该带有立场和情感倾向

  • 建立了健全，良好的关系再使用开放式问题，否则

    • 会被防御(窥探隐私，破坏关系)
    • 担心，疑惑，和对立

  • 通过提问收集信息，了解对方

  • 提问方式

    • 为什么(原因，过去的经验)
    • 怎么样，怎么想的(事实)
    • 开放式提问 = 详细回答
    • 封闭式提问 = 缩小范围，深层原因 和 引导话题到正题上

  • 带有主观，冷漠，责备的提问 vs 暖暖的关心

具体化

eg1:

  • 完全睡不着怎么办？ -> (你为什么睡不着？)觉得太无能了 -> (要不要去医院看看)嫌弃

eg2

  • 完全睡不着怎么办？ -> (完全睡不着？)翻来覆去，很晚睡 -> (你是入睡困难，不是完全睡不着)所以不是失眠？ -> (你入睡在想什么？)觉得太无能了,被人嫌弃 -> (有谁讨厌你，觉得你不好？)导师有批评/建议 -> (批评不等于嫌弃)批评导致压力大，害怕毕业困难

技术分析

  • 沟通中存在的问题(同名不同事物,认为的事实与对方认为的事实可以完全不同) -> 把问题具体化，明确化

  • 协助清楚，准确地表达观点和使用的概念，以及体验到的情感和经历的事情

  • 情景

    • 说出的问题模糊，喜欢含糊的，笼统的。烦死了
    • 过分概括，以偏概全，矛盾扩大，问题复杂。个例 -> 全部，事情->人，有时 -经常。过去 -现在，处处
      • 调整概括化的认知方式，具体的面对问题
    • 概念不清晰，澄清概念 你的意思是 能不能具体一点 你说的xx是指什么 能举个例子吗

  • 避免害怕别人觉得我不懂

影响性

做出反馈

内容表达

  • 内容表达(表达观点):提出建议，进行解释和反馈，促使行动
  • 注意措辞：希望你能/或许可以考虑

eg

  • 项目后期，没食欲，对伴侣烦恼(你不是喜欢这样？) -> 伴侣竟然提出结婚(结婚不是挺好) -> 不想结婚，吐槽伴侣在长辈面前说(这是对你的不尊重，你就是太好欺负了，应该当场说明态度) -> 沉默

技术分析

  • 竟然：说明没有准备，是意向不到的事
  • 当时如何如何：是马后炮，只会增加无力感和无助感
  • 群情激昂，可当事人面露难色：被指责

eg2

  • 项目后期，没食欲，对伴侣烦恼(你男朋友让你不开心了?) -> 伴侣竟然提出结婚(你是不是还不想结婚) -> 不想结婚，吐槽伴侣在长辈面前说(那你不很尴尬?) -> 吐槽不提前商量(不尊重你，应该先和你商量下，这是让你为难) -> 一点都不考虑我的感受(就是说，所以你很生气，也不商量下，那之后怎么办?) -> 太可气了(最可气的是不尊重你的感受，突然说结婚，突然当着长辈面说结婚，这才是最让人生气的吧) -> (主动问，你这么生气，你男朋友知道吗) -> 当然知道(那他知道你为什么生气吗) -> 你的意思是该和他沟通下

技术分析

  • 贴近真实感受，所以表达多，得到真实的态度

面质

  • 对矛盾的地方提出质疑，处于发生对峙的状态

  • 针对矛盾处，做工作，促进(对方)自我探索，内在统一 (不能把自己的主张作为统一目标)

  • 使用的原因：

    • 理想和现实不一致(想休息，但有很多工作)
    • 言行不一致(说沉迷游戏不好，又熬夜玩)
    • 言语不一致(结婚不好，但向往幸福婚姻)
    • 意见不一致(瘦子强调自己很胖)

  • 不是改变，而是促进

    • 对自己感受，信念，行为和环境的深入了解 (害怕焦虑导致看不到另一种感受，期待)
    • 促进放下防御，能够真实的面对自己，产生行动(微笑面对痛苦)
    • 促进言语和行为统一，理想和现实自我统一 (考清北和成绩中等)
    • 明确自己拥有的能力和优势，利用(留守家庭和已获得的幸福)
    • 树立模仿的榜样，使得对方有能力对自己进行面质

  • 如何把握面质的度

    • 以事实为基础
    • 避免个人情绪的宣泄变成攻击
    • 避免无情的攻击
    • 良好的咨询关系为基础
    • 使用尝试性面质(你上次似乎说……， 大概可能也许，恐怕可能不敢)
    • 留有余地，不要穷追猛打

eg

  • (为什么不结婚?) -> 结婚会出现各种问题(比如什么) -> 双方家长，孩子问题(这是普遍问题) -> 知道，但就是排斥(你难道不希望两人和谐？) -> 羡慕但遥远(别人能做到，你怎么做不到) -> (你的伴侣有给足安全感吗)有的 -> (那你在担心什么)害怕出轨 -> (这不是理由，对方有给你安全感) -> 不想说了

技术分析

  • 普遍问题存在，不等于可以否定焦虑的情感
  • 质问同时带有自己的情绪和指责 -> 失落+愤怒
  • 爹味，家长管制的意味 -> 反抗

eg2

  • (为什么不结婚?)-> 结婚会出现各种问题(比如什么) -> 双方家长，孩子问题(听上去很复杂) -> 是啊，结婚有很多的问题全出现(你是要一直和男朋友谈，不结婚?) -> 不一定(到时候你不怕担心的问题出现?) -> 不知道，就是担心(一说到结婚你想的全是担心吗？) -> 也向往幸福的婚姻(什么样的幸福婚姻) -> 两个老登坐公交(所以你强调不想结婚，其实还是向往幸福婚姻) -> 可是好难(难在哪里) -> 不接受出轨(当然了) -> 感情没到那个程度，害怕结婚让感情消磨(可是必须要做出选择) -> 喜欢你和你喜欢(重复) -> 担心选择(你害怕的是在感情中被抛弃)

技术分析

  • 不评判对错，肯定情绪 -> 被接纳 -> 提出永远不结婚的矛盾
  • 更温和的方式 = 表达疑惑(不理解，多说一点)
