会谈技巧

心理咨询的理论与方法：会谈技巧

  • 心理治疗 vs 心理咨询
  • 建立咨询关系，帮助人自己解决问题
  • 会谈技巧，信任，接纳

基本理念

尊重

  • 礼貌，姿态，不批评，不指责

  • 高中生厌烦学习

    • 批评指责不上进，理由是将来没出息 -> 没关注到焦虑的真正原因，和表现出的焦虑
    • 耐心询问为什么厌烦 ->
      • 怎么脸色有点差
      • 有学习压力是正常的
      • 你说xxx的意思是，具体化，+复述

  • 上大学是唯一出路吗？表现出想尝试不同的人生路

    • 数落 -> 仿佛扼杀邪念 -> 强加自己的价值观给别人
    • 开放，积极的态度下感到被理解，被接纳 -> 引导了解信息，付出努力
      • 确实xx（复述），你知道怎么实现吗？
      • 深入了解，实践也是xxx(复述)

  • 打断陈述并给出帮助

    • 随意打断并不礼貌 -> 一厢情愿的帮助 -> 居高临下的态度说教
    • 展现相同的境遇，婉转的提出帮助意愿 -> 激发主动合作的愿望
      • 情绪表达(辛苦，)
      • 自我开放

  • 随意泄露私下的谈话 ->

    • 隐私泄露
    • 笼统概括 -> 保护隐私

  • 良好的咨询关系

    • 理念态度+动机
    • 无条件尊重
    • 最大程度的表达自己
    • 自我价值感，恢复自信
    • 中性价值体系
    • 充分信任 - 信任动力
      • 求助者的认知最重要
      • 假话的意义
    • 尊重隐私
      • 评价-干预
